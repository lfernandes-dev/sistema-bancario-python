# Sistema Bancário em Python

Projeto simples feito em Python para praticar lógica de programação.

## Funcionalidades

- Ver saldo
- Sacar dinheiro
- Depositar dinheiro
- Menu interativo
- Submenu de atendimento

## Tecnologias

- Python

## Como executar

1. Baixe o projeto
2. Execute:

python banco.py
